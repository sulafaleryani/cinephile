# Cinephile Movie Platform

A single-file React application for discovering movies, managing a watchlist, and writing reviews using the TMDB API.

## Features

* **Trending Hero Banner:** Automatic slider showcasing current popular films.


* **Categorized Carousels:** Rows for trending, now playing, popular, upcoming, and top-rated movies.


* **Detailed Modals:** Tabbed views for movie summaries, complete cast lists, user reviews, and recommendations.


* **Live Search:** Input filtering with a built-in debounce to limit API requests.


* **Local Persistence:** Watchlist data and written reviews are saved directly to `localStorage`.



## Tech Stack

* React 18 (CDN)


* Babel Standalone


* TMDB API v3


* Vanilla CSS-in-JS



## Usage

1. Save the code into a file named `index-12.html`.
2. Open `index-12.html` directly in any web browser.
3. To use your own API credentials, update the `API_TOKEN` constant inside the script tag.
